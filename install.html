💻 Chromebook ・ Android
Chrome ブラウザ
  1. 1Chrome で QR を よみとる（または うえの URL を いれる）
  2. 2ゲームの がめんに でる ＋ ホームに ついか ボタンを おす
  3. 3「インストール」が でたら インストール を おす
  4. 4ホームに アイコンが できる → タップして あそぶ！🎮
かんたん！ ボタン 1かいで ホームに ついか できるよ。
📱 iPad ・ iPhone
Safari ブラウザ（さふぁり）
  1. 1Safari で QR を よみとる（または うえの URL を いれる）
  2. 2した（または うえ）の きょうゆうボタン ⬆️ を タップ
  3. 3＋ ホーム画面に追加 を えらぶ
  4. 4みぎうえの 追加 を おす → ホームに アイコン！🎮
iPhone / iPad は Safari で。きょうゆうボタンから ホームに ついか してね。
🏠 がくどう・おうちの かたへ：このゲームは むりょう・アプリストア不要の Webアプリ（PWA）です。ホームに ついかすると ふつうの アプリのように ひらけ、いちど ひらけば オフラインでも あそべます。Chromebook / Android は ボタン1つで、iPad / iPhone は Safari の「ホーム画面に追加」で セットアップ できます。
